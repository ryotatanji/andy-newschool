# andy-newschool

&Y オンラインビジネススクール 公式LINE（Anybot構築）プロトタイプ v3

## 概要
- 4入口 × 2主導線 × 5運用レイヤーを再現したLINE UI モック
- Anybot 挙動（リッチメニュー画像+ホットスポット、タグ付与、シナリオ分岐、セグメント配信、LIFFミニアプリ）を忠実に再現
- 単一 HTML ファイル（SVG描画、CSSレスポンシブ、Vanilla JS）

## 閲覧方法
GitHub Pages 公開後、以下のURLでアクセス可能：
`https://[GitHubユーザー名].github.io/andy-newschool/`

## ファイル構成
- `index.html` — プロトタイプ本体
- `.nojekyll` — GitHub Pages の Jekyll 処理を無効化
- `README.md` — このファイル

## 対応デバイス
- PC（Chrome / Safari / Edge 最新版）
- スマートフォン（iOS Safari / Android Chrome）
